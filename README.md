<h1>GENERATE REAL ESTATE LOT NUMBERING</h1>
<p>This tool aims to propose to the user the possibility of automatically generating the lot numbering, either in sequential numbering or by the measurement of its tested, in both methodologies, the user will need to define only the initial lot of each block so that the tool manage the other numbers, always clockwise.</p>

<h1>GERAR NUMERAÇÃO DE LOTE IMOBILIÁRIO</h1>
<p>Esta ferramenta visa propor ao usuário a possibilidade de gerar de forma automática a numeração do lote, seja ela em numeração sequencial ou pela medida da testada deste, em ambas as metodologias, o usuário necessitará definir apenas o lote inicial de cada quadra para que a ferramenta gere as demais numerações, sempre em sentido horário.</p>
